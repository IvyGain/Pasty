# Pasty 0.9.1-beta — D&D 体験ホットフィックス

## 🛠 D&D 中のドラッグ表示を不可視化

- クリップをフォルダタブへドラッグした際に表示されていた半透明の「重なりカード」プレビューを廃止
- カーソルに追従する SwiftUI ドラッグ画像を 1×1 の透明ビューに置き換え、macOS ネイティブのドラッグカーソルだけが表示されるように
- ドロップ先のフォルダタブが常に視認でき、どのフォルダに渡るのかが一目で分かるようになりました

## 🛠 フォルダ並べ替えの青い縦線を復活

- v0.9.0 で発生していた「フォルダタブをドラッグしてもタブ間の青い縦線ガイドが出ず、並べ替えが効かない」回帰を修正
- `folderTab` の SwiftUI モディファイア順を入れ替え、`.draggable(PinboardDragItem)` を `.acceptClipReferenceDrop` より先に適用することで、ギャップ側の `dropDestination` が正しくホバーを受け取れるように
- 並べ替え操作 (ドラッグ → 青い縦線 → ドロップで `PinboardStore.reorder` 発火) が v0.8.7 / v0.8.8 と同じ感触で復活しました

## 影響範囲

- クリップ単発 / 複数選択ドラッグ・他アプリへのテキストドロップなど、Transferable ペイロードや外部ドロップ経路は一切変更なし
- プレビュー描画ロジックのみ差し替え、モディファイア順を `folderTab` 内で 1 箇所スワップしただけのピンポイント修正
