# Pasty 0.10.0-beta — v1.0 準備リリース

## 🎯 ストレスゼロ UX 2

- 検索フィールドに 300ms キーストロークデバウンスを導入。タイピング中の FTS5 クエリ発火を約 70% 削減。
- FTS5 パターン LRU キャッシュ (64 件) で繰り返しクエリの CPU 負荷を 5–15% 軽減。
- ホバープレビューのキャンセル動作と空クエリ時の最適化を再検証。
- Swift 6 strict concurrency 準拠化: 対象 7 ファイルで警告 108 件 → 0 件、`@MainActor`/`Sendable` を明示化。

## 🌐 英語化

- Onboarding / Settings (一般タブ) / WhatsNew / Help を `Localizable.strings` (ja.lproj + en.lproj) に 70 キー追加 (en/ja パリティ済み)。
- 新ヘルパー `L10n(key)` / `L10nText(key)` / `L10nFormat.string(key, args...)` を導入。
- AI / Insights / OCR ラベルは v0.10.1 で追従予定。

## ♿️ アクセシビリティ

- 6 つの主要 View (StripPanel / SettingsWindow / OnboardingView / WhatsNewPresenter / HelpOverlayView / ClipPreviewView) に VoiceOver ラベル 30 件 + UI test 用識別子 2 件を追加。
- HelpOverlayView のショートカット行が「キー + 説明」を 1 発話に統合 — 30+ 行すべてが VoiceOver でクリアに読み上げ可能。
- Dynamic Type は v0.9.9 luxury UI/UX に視覚的回帰を起こさないため次回ターゲット。

## 📦 v1.0 準備

- DB GC: 90 日経過した soft-delete クリップを自動 hard-delete する `RetentionSweeper` を追加。起動時に BlobGC と並んで起動。
- マイグレーション統合テスト 3 件 + soft-delete ライフサイクルテスト 4 件 (計 7 件) で v1→v10 を網羅。
- Disk LRU: VideoThumbnailCache + PDFThumbnailCache に `~/Library/Caches/io.pasty.app/{video-thumbs,pdf-thumbs}/` 永続化 (40 MB + 20 MB / 90 日エビクション)。
- 本リリースで統合テスト 7 件 (マイグレーション v1→v10 / soft-delete ライフサイクル) を新規追加。`swift test` 51/51 グリーン。
