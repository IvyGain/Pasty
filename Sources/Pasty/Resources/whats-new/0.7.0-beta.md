# Pasty v0.7.0-beta — UX 改善 7 連発

「もっと早く、もっと静かに、もっとあなた色に。」

## ✨ 新機能

### 1. 検索/フィルタの記憶
ストリップを開き直しても、直前の検索クエリと種類フィルタが復元されます。
設定 → サーフェス → 「検索とフィルタ」でいつでも OFF にできます。

### 2. クイックペースト ⌃⌥1〜5
パネルを開かずに、直近 1〜5 件目のクリップを 1 アクションで貼り付け。
連続貼付が日常な人ほどに刺さるはずです。

### 3. 機密モード ⌃⌥⇧P
パスワードや個人情報を扱う前に 1 発で 60 秒間キャプチャを停止。
もう一度押せば即座に解除。トースト + サウンドで状態が分かります。

### 4. カスタム自動振分ルール
「特定ドメインを含む URL は『リサーチ』フォルダへ」「特定アプリからのコピーは
『仕事』フォルダへ」 — そんな個人ルールを最大 20 件まで登録できます。
デフォルトの 8 カテゴリ判定より優先されます。

### 5. リリースノートダイアログ
このダイアログ自体が新機能 :) アップデート直後に「何が変わったか」を
さっと確認できます。設定 → Pasty について → 「リリースノート」から
いつでも再表示できます。

### 6. AI マクロ (連鎖アクション)
複数の AI アクションを 1 クリックで連結実行。
プリセット例: 「英訳 → メール風」「要約 → メール風」など。

### 7. 節約時間カード
インサイトダッシュボードに「累計どのくらいの時間を節約したか」を表示。
1 回の貼付を 28 秒換算でざっくり可視化します。

## 🛠 内部改善

- Sparkle 自動アップデートのフィードバック改善
- ホットキー設定 UI に新アクションのアイコン追加
- 各種コードベース整理
