# Pasty 0.9.0-beta — 使い心地の完成形

## 🎯 複数選択 UX 完成形

- **ペースト完了で自動解除** — 1 回貼り終わるたびに選択がクリアされ、次に開いたときは常にまっさらな状態から始められます
- **Esc は 2 段階** — 多選択中は 1 回目で選択解除、2 回目でパネルを閉じる直感的な挙動 (通常モードは従来どおり 1 発で閉じます)
- **Enter で順番にまとめて貼付** — 多選択中に Enter を押すだけで、選んだ順番どおりに連続貼付されます
- **ドラッグでまとめて運ぶ** — 多選択中に 1 件をドラッグすると、選んだ全件まとめて他アプリやフォルダタブへドロップできます

## 📖 オンボーディング全面刷新

- 9 ステップ → 14 ステップに拡張。v0.8 系で追加された新機能 (ノッチホイール / フォルダドラッグ並べ替え / AI マクロ / 機密モード ⌃⌥⇧P / クイック貼付 ⌃⌥1-5) をすべてインフォグラフィックで案内
- 冒頭にカテゴリ別チップ (基本操作 / ノッチ / フォルダ / AI / ショートカット) を追加 — 必要なところだけ読めます
- KeyCap の押下アニメーション、SF Symbol の pulse / bounce を駆使した動きある案内

## ✨ 新機能ミニチュート

- リリースのたびに「今回追加された新機能」を 3-4 枚の小さなカードで自動表示
- WhatsNew のリリースノートと連動して、何が新しくなったかを忘れません

## 🖼 プレビュー崩れ修正

- 複数選択中に画像クリップのサムネがカード枠を超えてはみ出る不具合を解消

## 🔗 URL プレビュー

- リンククリップにタイトル + favicon + ドメインを表示する軽量プレビューを追加
- メモリ + ディスクキャッシュで高速、5 秒タイムアウト付き
- 設定 → サーフェス →「URL プレビュー (軽量フェッチ)」で OFF にも切り替え可
