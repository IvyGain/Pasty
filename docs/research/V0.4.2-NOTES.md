# Pasty v0.4.2 リリースノート

**バージョン**: v0.4.2 (build 10)
**テーマ**: SmartFolder 統合 + アクセシビリティ + 仕上げ

---

## 概要

v0.4.2 は v0.4.1 で導入された Intelligence / Templates / AutoCategorize / Settings の各機能を「**フォルダ体験**」を中心に有機的に結びつけるリリースです。SmartFolder の本格配線、Hover Preview のカード統合、永続化された Insights、設定タブのフル拡張、そして VoiceOver / Dynamic Type への正式対応により、Pasty は本格的な日常ツールとして完成度を一段引き上げます。

---

## 配線完了した機能

### 1. SmartFolder 7 builtin の StripView 統合

- **今日 / 今週 / 画像 / リンク / コード / 定型文 / 大容量** の 7 種類の動的フォルダを StripView 上部の **フォルダタブ** に常設表示
- タブは `ClipStore` のフィルタ述語と直結し、追加コストなしで切り替え可能
- 並び順・表示 / 非表示は設定タブから個別にトグル

### 2. スマートフォルダ「+ 保存」ボタン

- 検索バー横の **+ 保存** ボタンで、現在の検索クエリ・フィルタ条件をそのまま動的フォルダとして保存
- 保存後はフォルダタブに自動追加され、以後リアルタイムで条件に合致するクリップを集約
- 「`type:link 期間:7日 タグ:設計`」のような複合クエリも 1 タップでフォルダ化

### 3. Hover Preview Controller の StripCard 統合

- 既存の Preview Controller を **StripCard レベル**に組み込み、カーソルホバー 200ms で大きめプレビューを表示
- 画像 / リンク OG / コード ハイライト / 定型文プレースホルダ展開をすべてカード上で確認可能
- カード離脱でフェードアウト、Esc で即時クローズ

### 4. Spotlight モードに Explorer + AI hooks

- Spotlight 起動中も Explorer モード（フォルダ階層ビュー）に切り替え可能
- AI hooks（要約・分類提案・テンプレ変換）を Spotlight 内コマンドパレットから直接呼び出し
- 結果は元クリップに非破壊で適用、操作履歴は元に戻す可能

### 5. paste_events テーブル + ClipStoreContainer 経由の永続化

- 新規 SQLite テーブル `paste_events`（clip_id / pasted_at / source_app / target_app）を追加
- ペースト操作はすべて `ClipStoreContainer` を経由して同期書き込み
- 既存 `clips` テーブルとは独立し、削除済みクリップの利用統計も保持

### 6. InsightsDashboard「最も再利用 Top 10」

- `paste_events` を集計し、過去 7 日 / 30 日 / 全期間で **最も再利用されたクリップ Top 10** を表示
- 再利用回数・最終利用日・利用先アプリ分布をミニチャートで可視化
- 1 クリックでそのクリップを定型文化 / ピン留め可能

### 7. AutoCategorizer のカテゴリ → フォルダ マッピング UI

- 設定タブに **カテゴリ → フォルダ** のマッピング表を追加
- AutoCategorizer が推論したカテゴリ（image / link / code / quote …）を任意のフォルダに自動振り分け
- マッピング未設定カテゴリは「未分類」フォルダにフォールバック

### 8. HotkeyStore からの動的グローバルキー登録

- `PanelCoordinator` が起動時 / 設定変更時に `HotkeyStore` を購読
- ユーザーが追加したカスタムホットキー（フォルダ切替・テンプレ呼び出し等）を**再起動なしで**グローバル登録
- 衝突検出 UI を設定タブに併設

### 9. FieldHistoryStore による `[[name]]` サジェスト

- 定型文の `[[name]]` プレースホルダ入力時に、**過去の入力値**を頻度順でサジェスト
- フィールド名ごとに履歴を分離保存、TTL（30 日）と最大件数（50）で自動掃除
- 設定タブから履歴クリア / エクスポート可能

### 10. 設定タブ拡充：プレビュー / フィードバック / AI

- **プレビュー**: ホバー遅延 / カードサイズ / OG 画像読み込みのトグル
- **フィードバック**: 触覚 / 効果音 / トースト表示の細粒度制御
- **AI**: 自動分類・要約の有効化、ローカル / API プロバイダ切替、APIキー管理
- 既存タブと統一されたデザイントークン（v0.4.1 から継続）

### 11. アクセシビリティ強化

- **VoiceOver**: StripCard / フォルダタブ / Spotlight 全てに `accessibilityLabel` / `accessibilityHint` を付与
- **Dynamic Type**: 本文サイズに追従する SwiftUI `.dynamicTypeSize` を全主要ビューに適用（xSmall ～ xxxLarge）
- キーボードフォーカスリングを Tab / Shift+Tab で循環、フォーカスインジケータも強調表示
