Pasty v0.9.8-beta

Pasty v0.9.8-beta — 静かに、自動で、ラグジュアリーに

「ノイズを消し、上限を自動で守り、見た目を一段引き上げる。」v0.9.7-beta の P1 一掃に続き、Wave 1–2 で UX とビジュアルを総ざらいします。

🔇 静かに

  • 1000 件超過時のトースト通知を撤廃しました
  • 検索結果上限の警告も非表示化しました
  • バックグラウンドのパフォーマンス最適化 (LIMIT 1000) はそのまま継続します

🧹 自動 trim 設定

  • クリップ数が上限を超えた際に自動で古いクリップを soft-delete する機能を追加
  • ピン留めしたクリップは保護されます
  • 設定 → 一般 → クリップ上限 で有効/無効と上限値を選択可能 (500 / 1000 / 5000 / 10000 / 無制限)
  • 起動時と新規クリップ追加時 (60 秒スロットル) に自動実行

💎 ラグジュアリー UI/UX

  • 統一デザイントークン (PastyDesign) を全画面に適用
  • カード/パネルに多層シャドウ + マテリアル backdrop
  • フォルダタブを Apple Wallet 風グラデーション + soft shadow に
  • HoverPreview にアクセントストリップ + prominent shadow
  • 設定画面の typography とタブバーを刷新 (ultraThinMaterial + accent underline)
  • WhatsNew + Onboarding に hero gradient + ステップインジケータ追加
  • ライト/ダーク両モード対応 (NSAppearance dynamicProvider)
