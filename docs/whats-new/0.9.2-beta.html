Pasty v0.9.2-beta

Pasty 0.9.2-beta — フォルダドラッグの視認性ホットフィックス

🎯 フォルダドラッグ中のタブを半透明に

  • フォルダタブを掴んで並べ替えるとき、掴んでいるタブが opacity 0.2 まで薄くフェードするようになりました
  • これによりドロップ先 (タブ間の青い縦線インジケータ) が見やすくなり、目的のスロットを正確に狙えるように
  • ドロップ完了時もキャンセル時 (Esc / ウィンドウ外でリリース) も、opacity は確実に 1.0 に復帰します

影響範囲

  • フォルダタブ並べ替え (PinboardDragItem) のドラッグ表示のみ変更
  • クリップドラッグ (ClipDragItem / MultiClipDragItem) はフォルダタブをフェードさせません — v0.9.1 と同じ挙動
  • v0.8.7 以降の青い縦線インジケータ / folderDropGap.dropDestination ロジックには変更なし
