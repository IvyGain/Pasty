why pasty

Built on the things Paste forgot.

Pasty studied every complaint left on Paste's review pages, Reddit, and Hacker News. Then it answered them.

⚡

Lighter than Maccy.

Idle under 60 MB. Cold launch in under 0.2 s. Polling done with care, not panic — and the 2.3 MB dmg proves it.

🔒

Local. Period.

Nothing leaves your Mac. No cloud account, no telemetry, no analytics. The store is a single SQLite file you can inspect any time.

⌥

The notch knows.

The first clipboard manager that turns the notch into a verb. Hover to summon, drag to paste, no keyboard required.

privacy by default

Your clipboard is yours. Yours alone.

Pasty stores everything in a single SQLite file under your Application Support folder. Pause it, wipe it, peek inside with the macOS Terminal. There is nothing else.

Zero networking.
Zero analytics.
Zero secrets to lose.

The kindest thing a clipboard manager can do is be boring on the network. Pasty's binary has no HTTP client wired up at all. Sensitive flag on the pasteboard? Skipped. Password manager up front? Skipped. You can blacklist apps by bundle id too.

⌃⇧P pauses capture for 60 seconds. The Settings → Privacy tab will wipe everything in one click.

how to use

Five minutes to never lose a copy again.

Pasty is designed to vanish into the muscle memory you already have. There are no tutorials to sit through, no accounts to make. Here is everything, in order.

1

Install & first launch

Download the dmg. Drag Pasty.app to /Applications. The first time, right-click → Open to dismiss Gatekeeper. The menu-bar icon appears in your top-right.

2

Grant Accessibility

Pasty asks once. This permission is what lets it post ⌘V into the app you were just using. Nothing else uses it.

3

Copy as you normally would

Use ⌘C, drag-to-copy, Share Sheets — anything. Pasty captures it in the background and dedupes by content hash.

4

Press ⇧⌘V

The Spotlight-style modal appears at your cursor's screen. Type freely or pull out the DSL (type:link source:Safari /TODO/). Hit Return. Done.

5

Or press ⌥⇧V

The bottom strip slides up. Use ←/→ to scan, ⌘1-9 to paste a specific card, ⇧↩ to strip rich-text formatting on the way in.

6

Hover the notch

Float your cursor over the MacBook notch. A Liquid-Glass strip slides down. Drag any card onto your editor — instant paste, mouse only. (Non-notched Mac? The top-centre 20% becomes a virtual notch.)

7

Pin to a Pinboard

Right-click any clip → "Pin to Inbox / Work / Code / Refs". Each pinboard is colour-tagged. Make new ones from Settings → Pinboards.

8

Stack & paste in order

Add multiple clips to a Stack, then walk through them with repeated ⌘V. Useful for forms, transcripts, multi-cell tables.

9

Pause it any time

⌃⇧P pauses capture for 60 seconds — perfect for when you're juggling a password manager. The menu-bar icon turns coloured while paused.
